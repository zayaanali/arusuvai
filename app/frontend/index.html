Manual Commands

Pantry Basics

  • add item1,item2 - Add all comma-separated items
  • rm item1,item2 - Remove matching pantry items
  • list - Show all pantry items in snapshot
  • drop_table - Clear entire pantry

Inventory Updates

  • set item,qty,category - Set quantity and category
  • use item,qty - Subtract quantity
  • discard item,qty - Subtract quantity (same as use for now)
  • inc item,qty - Increase quantity
  • dec item,qty - Decrease quantity

Metadata

  • rename old,new - Rename item
  • category item,cat - Update category
  • expiry item,YYYY-MM-DD - Update expiration date

Search and Views

  • find term - Filter pantry by item name
  • expiring N - Show items expiring within N days

Shopping List

  • shopping add item1,item2 - Add shopping items
  • shopping rm item1,item2 - Remove shopping items
  • shopping list - Show shopping list

About arusuvai

Arusuvai (அறுசுவை) means “six tastes” in Tamil — the idea that a complete, satisfying meal balances sweet, sour, salty, spicy, bitter, and astringent flavors into one harmonious whole.
